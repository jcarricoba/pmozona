Dar de alta un proyecto

1
Pulsa + Proyecto en la barra lateral para abrir el formulario.
2
Rellena los campos obligatorios: Cliente y Nombre del proyecto.
3
Elige el Estado inicial, el Area y el Responsable.
4
Introduce las Jornadas estimadas, las Jornadas consumidas y, si aplica, las Jornadas de externos (subcontrata). El % de progreso se calcula solo.
5
Opcional: define el presupuesto por perfiles con Editar presupuesto, y añade URLs de seguimiento, oferta o plan.
6
Si usas partes de trabajo, anota el Codigo de proyecto (ej. PRO-0145-01) para vincular las horas automaticamente.
7
Guarda. El proyecto aparece en el Portfolio y los KPIs se actualizan al momento.

Mantener un proyecto

1
Haz clic en cualquier fila del portfolio para expandirla y ver el detalle completo.
2
Pulsa el icono de edicion para actualizar jornadas. El % progreso se calcula automaticamente desde jornadas estimadas, consumidas y de externos. Marca Fijar manualmente si quieres introducir un valor propio.
3
En la pestana Comentarios de la ficha puedes añadir notas de seguimiento con fecha y autor.
4
Cambia el estado directamente haciendo clic en el punto de color de la fila.
5
Cuando el proyecto termine, usa Archivar para moverlo al Historico con las jornadas reales y lecciones aprendidas.

Estados del proyecto

Dentro del plan Todo correcto, avanza segun lo previsto.
Planificado En preparacion o pendiente de arrancar.
Bloqueado Hay un impedimento que impide avanzar.
Fuera del plan Desviacion significativa en plazo o alcance.

Partes de trabajo

1
Exporta el Excel de partes desde la app interna de gestion de horas.
2
Ve a la seccion Partes en la barra lateral y pulsa Importar Excel.
3
En la ficha de cada proyecto, escribe el Codigo de proyecto que aparece como nombre de hoja en el Excel (ej. PRO-0145-01).
4
Las horas consumidas se actualizan automaticamente. Aparece el icono 📋 en la columna de jornadas cuando el dato viene de partes. Las jornadas de externos se suman encima.
5
Los partes se mantienen en memoria: hay que reimportarlos si recargas la pagina.

Preguntas frecuentes

Como cambio entre jornadas y horas?

Pulsa el boton Jornadas / Horas en la barra lateral. Todos los valores del portfolio se convierten automaticamente usando el factor configurado en Ajustes (por defecto 8 h/jornada).

Como funciona el calculo automatico de progreso?

El % se calcula como (jornadas consumidas + jornadas de externos) / jornadas estimadas. Si hay partes importados vinculados al proyecto, las horas de partes sustituyen a las jornadas consumidas manuales. Activa Fijar manualmente en la ficha si necesitas introducir un valor distinto.

Donde se guardan los datos?

En el localStorage de tu navegador. Los datos no salen de tu equipo salvo que configures la sincronizacion en la nube (JSONBin) desde Ajustes. Exporta regularmente en JSON como copia de seguridad.

Puedo tener varios espacios de trabajo?

Si. Usa Exportar para guardar el estado actual como fichero JSON, carga otro fichero con Importar. Cada fichero JSON es un workspace independiente. Al importar, el Bin ID viene siempre del propio fichero — si el fichero no tiene bin, se deja vacio para no mezclar datos de distintos workspaces. La API Key puede heredarse de Ajustes.

Como funciona el presupuesto?

Dentro de la ficha de proyecto, pulsa Editar presupuesto para añadir lineas por perfil (ej. 3 jornadas de Consultor Senior a 95 €/h). El coste estimado y el margen se calculan solos y aparecen en el portfolio y en los KPIs agregados.

Que pasa si borro el navegador o cambio de equipo?

Perderas los datos si no has exportado ni activado la nube. Exporta el JSON periodicamente o configura JSONBin en Ajustes para tener una copia accesible desde cualquier sitio.

Como añado un stakeholder o documento a un proyecto?

Edita el proyecto y ve a las pestanas Stakeholders o Documentos dentro del formulario. Puedes añadir contactos con rol, email y telefono, y enlazar documentos con URL.
